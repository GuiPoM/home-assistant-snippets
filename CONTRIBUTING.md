# Guide d'Utilisation

## Comment utiliser ce repository

### Pour suivre un tutoriel

1. Clonez le repository
   ```bash
   git clone https://github.com/GuiPoM/home-assistant-snippets.git
   cd home-assistant-snippets
   ```

2. Consultez la branche vidéo correspondante
   ```bash
   git checkout video/ikea-klippbok
   ```

3. Lisez le README de la branche pour les instructions complètes

## ⚠️ Importante Notice

Ce repository est fourni à titre informatif pour accompagner les tutoriels vidéo YouTube. 

**Redistribution et contributions ne sont pas acceptées.** Ce code est destiné à être utilisé personnellement pour votre propre installation Home Assistant.

## Structure d'une branche vidéo

```
video/nom-de-la-video/
├── README.md              # Instructions et liens
├── blueprints/            # Fichiers blueprint
├── scripts/               # Scripts Home Assistant
├── custom_templates/      # Macros Jinja
└── examples/              # Exemples d'utilisation
```
