Une collection de blueprints Home Assistant et scripts Jinja avec des tutoriels YouTube accompagnants.

🎥 **Découvrez nos tutoriels vidéo et retrouvez tout le code source associé !**

## 🚀 Guide de Démarrage

1. **Choisissez un tutoriel vidéo** qui vous intéresse
2. **Consultez la branche correspondante** pour accéder au code complet
3. **Suivez le README** de la branche pour les instructions détaillées

## 📁 Structure

- **Main branch** : Vue d'ensemble et liens vers tous les tutoriels
- **Branches vidéo** (`video/*`) : Code complet, documentation et exemples pour chaque vidéo
- Chaque branche est **indépendante** et peut être explorée séparément
